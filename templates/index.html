About ShopStop

CS432 Module 3 - Database Integration and Secure API Development

ShopStop is a comprehensive retail management system that connects project-specific databases to a central database (CIMS - Central Information Management System).

Key Features:
  • Secure Authentication: Role-based access control with session management
  • Member Management: Create, update, and delete members with proper integrity checks
  • Shop Management: Manage shops, products, and inventory
  • Customer Management: Track customer information and purchase history
  • Portfolio Management: View and update member portfolios
  • Comprehensive Logging: Track all database changes for security and audit purposes
